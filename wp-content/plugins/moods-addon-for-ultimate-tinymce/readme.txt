== Description ==
Add over 50 animated smilies to your visual tinymce editor.  

This plugin is designed to be used as an add-on to my Ultimate TinyMCE plugin.  However, it can also be used as a stand-alone plugin as well.

For best results; You can download Ultimate TinyMCE here:<br />
http://wordpress.org/extend/plugins/ultimate-tinymce/

This addon will add a button to your visual tinymce editor for posts/pages.  Clicking the button will open a popup window with over 50 professionally animated .gif smiley's.

You can insert these smiley's into your post/page content areas.  Simply click a smiley, and it is automatically inserted into your content area.



== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
     
2. Activate the plugin through the 'Plugins' menu in WordPress.

3.  That's it!!  Go to your post/page editor and enjoy the Moods.

Note:  If you have my Ultimate TinyMCE plugin installed, you can go to it's admin settings page and there is a box letting you know if the addon is activated or not.


== Frequently Asked Questions ==

= I don't see the moods icon once the plugin is activated. =
Make sure to click the "show/hide kitchen sink" button in the top row of the editor.  This will expand the editor, showing a second row of buttons.
